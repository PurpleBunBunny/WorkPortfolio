GripApp

An App For Personal Trainers.

Grip is a Fullstack application that allows personal trainers to track their clients and their progress. It includes user authentication using passport.js and bcrypt, data persistence using MongoDB, data retrieval via Mongoose, Front-end UI/UX using React Native, graphs assembled using React Native Chart Kit, and finally utilizes an Express server built with Node.js.

  • Date: February 2019
  • Clientele: Personal Trainers/Fitness Workers
  • Category: Fitness/Mobile App
  • GripApp
  • GripApp Backend

Take A Hike

A Web App to find local Seattle Hikes.

This app is something we created to get info on trails in the seattle area. It will provide a quick list of things you might want to know about a hike. People are also able to comment about any hike on the map, all you have to do to learn about a hike is to click on a marker. You can tell how difficult a hike is by the icon it has on the map

  • Date: December 2018
  • Clientele: Outdoor Enthusiast
  • Category: Hiking/Web App
  • Take-A-Hike

Bamazon

A backend shopping app for buyers and sellers.

This app is solely a backend app that emulates shopping sites like amazon, in which shoppers can serach for and buy items they are interested in. It also has a fully functional seller side, in which sellers can add items that they'd wish to sell into the public market. Managing stock of items and other basic managerial aspects are included in the app.

  • Date: November 2018
  • Clientele: Buisness/Sellers
  • Category: Shopping/ Web App
  • Bamazon

Trivia Game

A simple timed Trivia Game using JavaScript.

A trvia game that uses JavaScript, HTML, and CSS as its base to create a simple question/answer game. When a user answers a question, they are are alerted with a response of correct or incorrect based on their choice. The game calcuates the total answers at the end of the game. If the user fails to finish all the questions, under the alloted time frame the game also ends.

  • Date: September 2018
  • Clientele: Gamers/Trivia Fans
  • Category: Games/Web App
  • Trivia Game

Sports Friend Finder

A app to help find you sports friends.

This app incorperates both frontend and backend code to help users connect and interact with other users that have similar likes and dislikes. The app starts you out by making you answer some survey questions to help with the matching system, then once you finish the app, it will show you who your best sports friend match is.

  • Date: November 2018
  • Clientele: Social/Sports/Dating
  • Category: Dating/Social/Web App
  • Sports Friend Finder
